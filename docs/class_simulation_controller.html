Detailed Description

Manages the execution and state of simulations, coordinating events and updates.

This class is responsible for managing the overall simulation process, including starting, stopping, and updating the simulation according to user inputs or predefined scenarios. It acts as an event subscriber to respond to various simulation events and control the simulation accordingly.
