Detailed Description

Wraps std::thread to ensure threads are joined on destruction, preventing dangling threads.

This class provides a wrapper around std::thread to manage the lifecycle of thread objects in a safe manner. It ensures that each thread is joined before the ThreadWrapper object is destroyed, thereby avoiding any potential for leaving unjoined threads that might execute beyond the lifetime of the objects they were intended to serve.
